## How-to

1. clone the repo or download zip file
2. edit .env.sample and rename to .env
3. edit server/db/knexfile.sample.js and rename to knexfile.js
4. run 'npm install'
5. run 'npm run watch' (nodemon) or 'npm run start' (node)
